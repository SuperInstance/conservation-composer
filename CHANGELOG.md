## [0.1.0] - 2026-06-08

### Added
- Interactive browser-based conservation law composer
- Real-time graph visualization with spectral analysis
- Conservation ratio (CR) meter and style classification
- Fiedler vector visualization
- Eigenvalue spectrum display
- Preset system (star, complete, jazz ii-V-I, random chaos)
- WAV export functionality
- Spectrum bar visualization
