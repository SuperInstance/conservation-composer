# conservation-composer

Interactive browser-based conservation law composer — visualize, explore, and compose with conservation principles in a real-time graph-based interface.

## What This Gives You

- **Real-time graph visualization** — interactive canvas for exploring conservation relationships
- **Spectrum analysis** — live spectral bar visualization of composed systems
- **Preset system** — pre-built configurations for common conservation patterns
- **Zero dependencies** — single HTML file, runs in any browser

## Quick Start

```bash
# Just open it
open index.html
# Or serve it
python -m http.server 8000
open http://localhost:8000
```
